# Teaching Design System (计算机网络试讲教学设计系统)

中职计算机网络教师招聘 10 分钟无生试讲 Codex Skill。根据教材内容自动生成教案骨架和 950-1100 字考场逐字稿，覆盖内容选取、教学设计、逐字稿扩写、审查和修订的完整闭环。

---

## 核心设计理念与特色

一切围绕考场实战。本系统杜绝机械的流程灌输，强调：
- **工程与常识优先**：在构建教案与话术逻辑时，绝不盲目顺从教材字面排版，而是引导虚拟学生根据真实网络工程实践进行逻辑推演。
- **显性点明重难点**：在导入过渡或重点讲授前，教师话术会显性向学生（评委）强调“这是本节课的核心重点/最大难点”，起到引起注意的作用。
- **极简巩固练习**：将练习时间压缩在 **45秒内**（采用快速判断、场景选择或简答），快速收束，把更多宝贵时间留给新课讲授。
- **极净客体化排版**：逐字稿中禁止出现 `[面带微笑]` 等动作标签，复述学生回答时采用“客体化复述”（不用第二人称“你”，而是使用“这位同学”）。

## 部署与安装指南

本 Skill 适配 Codex，其他人可以通过以下两种方式一键安装：

### 方法一：通过 GitHub 远程一键安装 (推荐)
在 Codex 聊天界面中直接对 Codex 发送安装指令：
```text
/install-skill https://github.com/guotaogithub/Teaching-Design-System
```

### 方法二：本地手动安装
1. 克隆本项目到本地：
   ```bash
   git clone git@github.com:guotaogithub/Teaching-Design-System.git
   ```
2. 将克隆下来的文件夹复制到 Codex 技能路径下：
   ```bash
   mkdir -p ~/.codex/skills/teaching-design-system
   cp -r Teaching-Design-System/* ~/.codex/skills/teaching-design-system/
   ```

安装完成后，在新建的 Codex 聊天中输入 `$` 即可自动联想并加载 `$teaching-design-system`。

## 快速使用 (Prompt 示例)

安装 Skill 后，你可以直接通过以下示例指令来启动对应的工作流：

### 1. 全套教案与逐字稿生成 (最常用)
> **Prompt**:
> @teaching-design-system 帮我设计一节新课。
> 课题是：**《计算机网络的拓扑结构》**
> 教材片段内容如下：
> [在此粘贴你的教材文本或扫描出的图片文字]

### 2. 只生成教学设计骨架 (Lesson Design)
> **Prompt**:
> @teaching-design-system 只生成教学设计骨架，说明内容取舍依据。
> 课题是：**《IP地址与子网掩码》**
> 包含知识点：IP地址分类、子网掩码的作用、子网划分计算。

### 3. Review 现有试讲稿并修订
> **Prompt**:
> @teaching-design-system 帮我 Review 并且修改下面这段 10 分钟试讲逐字稿，看看时间、互动、重难点有没有问题：
> [在此粘贴你写好的逐字稿]

## Skill 功能路由说明

本 Skill 支持 5 种任务模式，根据用户请求自动选择最小必要流程，避免上下文过度膨胀：

1. **创建新课程**：提供课题或教材片段 → 内容选取 → 教学设计骨架 → 逐字稿 → 自检修订
2. **只生成教学设计**：提供材料 → 五环节教学设计骨架 + 内容取舍说明
3. **只生成逐字稿**：提供已有教案 → 扩写 950-1100 字逐字稿 + 字数统计 + Checklist
4. **Review 现成逐字稿**：提供逐字稿 → 按标准诊断 → 分级问题 + 修改建议
5. **修订现成逐字稿**：提供逐字稿 → 诊断 → 修正 → 最终稿 + 修改说明

## 目录结构

- `SKILL.md`：Skill 入口，含渐变披露规则和策略文件映射
- `references/`：Skill 执行内核，按需加载以节省 token 消耗
  - `standard-core.md` — 基础总规（时间、容量、结构、互动、语言等）
  - `content-selection.md` — 内容选取与轻量串联/合理留白策略
  - `lesson-design.md` — 教学设计骨架生成与策略选择（重难点与练习时间已适配新标准）
  - `script-writing.md` — 逐字稿扩写规则与互动写法（新增显性重难点话术指引）
  - `script-review.md` — 逐字稿审查清单与严重度分级
  - `script-revision.md` — 逐字稿修订工作流与常见修改策略
- `Strategy-Library/`：9 个结构化策略文件，覆盖导入、提问、互动、讲解、过渡、板书、练习、评价、收束
  - `Practice_Strategy.md` (已升级为 V2，主打快速判断/简答，限制在 45 秒内)
- `Prompts/`：仅保留 V2.0 原始母版，用于追溯对照
- `Courses/`：优质中职课例存档（包括数据传输方式、拓扑结构、网络病毒防范等）
