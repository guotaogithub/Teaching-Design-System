# Introduction Strategy

| ID | Strategy | Description |
|----|----------|-------------|
| INT-001 | 问题悬念导入 | 提出具有悬念或认知冲突的问题，激发学生求知欲 |
| INT-002 | 生活情境导入 | 创设熟悉的日常真实生活情境，拉近知识与现实距离 |
| INT-003 | 温故知新导入 | 简要回顾已学相关旧知，通过逻辑延伸自然过渡到新课 |
| INT-004 | 故事逸闻导入 | 讲述与课题相关的历史典故或人物故事，吸引注意力 |
| INT-005 | 游戏互动导入 | 设计高度契合主题的简短互动游戏，迅速活跃课堂氛围 |
| INT-006 | 直观教具导入 | 直接展示实物、模型或挂图，给予学生直观的视觉冲击 |
| INT-007 | 实验演示导入 | 通过现场演示奇特的实验现象，引导观察并引发思考 |
| INT-008 | 多媒体视听导入 | 播放切合主题的视频、动画或音频，快速渲染学习氛围 |
| INT-009 | 时事热点导入 | 结合最新新闻热点或社会关注现象，引发共鸣与探讨 |
| INT-010 | 典型案例导入 | 抛出具有代表性的具体案例供剖析，直接切入核心议题 |
| INT-011 | 角色扮演导入 | 安排学生模拟特定身份体验预设场景，在实操中发现问题 |
| INT-012 | 数据图表导入 | 展示具有震撼力或反差强烈的统计数据，迅速引起重视 |
| INT-013 | 趣味猜谜导入 | 采用猜谜语或脑筋急转弯形式，以趣味方式引出核心概念 |
| INT-014 | 名言警句导入 | 引用哲理名言或经典诗词，提升课堂格调并切入主题 |
| INT-015 | 类比联想导入 | 借助学生熟知的事物属性进行类比，降低新知理解门槛 |
| INT-016 | 任务驱动导入 | 开篇直接下发具体的闯关任务，引导学生带着目标学习 |
| INT-017 | 误区纠错导入 | 展示常见的错题案例或常识误区，打破原有固化认知 |
| INT-018 | 竞赛比拼导入 | 组织简单的分组知识抢答或技能比拼，点燃学习热情 |
