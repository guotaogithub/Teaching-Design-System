# 核心规范与致命红线（严格执行，违者淘汰）

## 一、 课堂结构与动作指令纯净度
1. **五大版块切分**：逐字稿必须包含且仅包含以下五个环节，并在段落前加上相应的**加粗标题**：
   - **【一、 导入新课】**（结尾必须出现 `[板书课题：XXX]`）
   - **【二、 新课讲授】**（分重点推进）
   - **【三、 巩固练习】**（体现教师的巡视与引导纠错）
   - **【四、 课堂小结】**（优先请学生总结，教师补充，并自然融入20-40秒的**课程思政**）
   - **【五、 布置作业】**（必做基础 + 选做拓展，结尾简洁无拖尾）
2. **零废话动作**：绝对禁止输出关于停顿、微笑、走动的动作标记。全篇唯一保留的提示符是 **`[板书：XXX]`**，且必须贯彻“随讲随写、只写关键词”的原则。

## 二、 引导式教学与工程逻辑（杜绝填鸭灌输）
1. **工程常识优先**：在讲解防毒、拓扑等具层级概念时，必须遵循现实世界由外到内、自下而上等数据流真实逻辑，不可盲目照念教材排版。
2. **四步建构法**：核心概念绝对不允许由教师直接念出定义。必须走四步：抛出现象 → 学生回答 → 教师顺势贴标签（概念命名） → 验证推演。

## 三、 师生互动的客体化与具体评价（语感高压线）
1. **统一动作闭环**：学生回答后必须走：示意请坐 → 针对性评价 → 复述提炼 → 推进。
2. **客体化复述**：复述学生答案时，**严禁使用第二人称“你”**。必须面向全班使用第三人称，如“正如他刚才提到的...”、“他们小组分析得很有逻辑...”。
3. **禁止水词评价**：禁止说“很好/非常棒”。评价必须具体，如“概括得比较完整”、“分析比较准确”、“提炼出了关键词”。

## 四、 时间与极限字数底盘
- 教师实际说话字数必须**死死卡在 950～1100 字之间**。砍掉冗长解释，将字数分配给高质量的启发与互动。
