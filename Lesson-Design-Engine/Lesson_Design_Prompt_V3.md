# Strategy Library 调用规范
在生成教学设计时，你**必须且只能**从提供的 Strategy Library 知识库中挑选策略，并在对应环节注明【调用的策略名称与 ID】：

1. **导入新课 (Introduction)**: 必须从 `Introduction_Strategy.md` 选取（控制在 1 分钟左右）。
2. **新课讲授 (New Knowledge)**: 必须从 `Explanation_Strategy.md` 或 `Interaction_Strategy.md` 中选取，针对每一个重点知识进行策略挂载。
3. **巩固练习 (Practice)**: 必须从 `Practice_Strategy.md` 或 `Interaction_Strategy.md` 中选取，覆盖本节重点。
4. **课堂小结 (Summary)**: 必须从 `Closing_Strategy.md` 中选取。
5. **布置作业 (Homework)**: 必须采用“必做+选做”分层设计。
6. **贯穿全程**: `Questioning_Strategy.md`（提问）与 `Transition_Strategy.md`（过渡）、`Blackboard_Strategy.md`（板书）必须在环节内灵活调用。

# 核心动作约束（服务于逐字稿生成）
- 在预演师生互动时，必须体现标准动作流：**“学生回答 → 教师示意请坐 → 教师复述（提炼） → 自然推进教学”**。
- 在教学设计中必须留出 20-40 秒的**课程思政融入点**（安全意识、工程素养等）。
